Program Name: Threaded Message Queue

Commnads To run:
make
./test

About:
This program uses a signle message queue that is control by threads.  Each time a new messgage is to be displayed a thread join is called.
The message is then passed the the thread the the thread controls the output and printing to the xml file.  There are no 
known errors to report at this time.  The xml outout can be found the outputLog.xml.   

Program And Data:
The program has already been complied and ran you may look at the outputLog.xml and see that it contains data.  If you choose to compile
this program comes with a complte make file so just type make and then ./test to run the exe.
